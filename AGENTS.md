# AGENTS.md

奈良県履物協同組合連合会 公式サイトのリポジトリ。`index.html` の 1 ページ構成、`assets/css/style.css` ＋ `assets/js/main.js`。GitHub Pages（`main` ブランチ）で公開。

## テーマカラー: ならはきものネイビー

サイトのブランドカラーは濃紺（ネイビー）系のパレット **「ならはきものネイビー」**。白・黒・赤を基調とした配色に調和するよう選定。色は `assets/css/style.css` の CSS 変数で一元管理しており、ライト／ダーク両モードで値を切り替える。**新しい色をハードコードせず、必ず以下の変数を使うこと。**

| 変数 | ライトモード | ダークモード | 主な用途 |
|---|---|---|---|
| `--primary-color` | `#1F3A5F` | `#6B9BD1` | 見出し下線、ボタン枠線、ホバー効果 |
| `--secondary-color` | `#3A5A87` | `#8AAFD8` | 補助的なアクセント |
| `--accent-color` | `#5E7BA6` | `#A7C0DF` | 軽いアクセント |
| `--link-color` | `#1F3A5F` | `#7BA3D9` | 組合員プロフィール等のリンク |

赤系のアクセント（`--gradient-text`: 黒→赤 / 白→赤）は見出しの装飾に使用しており、ネイビーと併用する想定。

## 実装上の規約

- **国際化**: 全コンテンツは日本語・英語の bilingual。`lang="ja"` / `lang="en"` 属性を付けた要素を併記し、`html[lang="..."]` セレクタの CSS で表示を切り替える。新規コンテンツも必ず両言語のペアで追加すること。
- **ブランチ運用**: `develop` で作業 → コミット → `main` へ fast-forward マージ → 両ブランチを push。公開は `main` への push で反映（GitHub Pages のビルドに 1〜2 分）。
